# Local JavaScript Exercise Testing

This context describes running 01-edu JavaScript exercise tests against a
learner's local solution without requiring the official Docker image.

## Language

**Exercise**:
A named 01-edu JavaScript task whose vendored tests can be run locally.
_Avoid_: Challenge, problem

**Solution**:
A learner's JavaScript or ECMAScript module file for one exercise.
_Avoid_: Submission, answer

**Solution directory**:
The directory containing one or more solutions, each named after its exercise.
_Avoid_: Student path, project folder

**Local run**:
One execution of an exercise's vendored tests against a solution outside the official Docker image.
_Avoid_: Test job, invocation
